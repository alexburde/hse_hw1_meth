# 1. Aнализ QC прочтений 
Какие особенности можно наблюдать по сравнению с секвенированием ДНК или РНК? 
Основной метод, направленный на изучение паттерна метилирования ДНК — бисульфитная конверсия, позволяющая отличить метилированные цитозины от неметилированных. Данное секвенирование приводит к превращению неметилированных остатков цитозина в остатки урацила, а метилированные цитозины остаются без изменений. Преобразованный таким образом цитозин распознается как тимин при последующей амплификации ДНК методом ПЦР. Как мы видим из графика, при бисульфидном секвинировании содержание каждого из нуклеотидов разное. Самое высокое значение достигается для тимина, а самое низкое для цитозина:

![image](https://user-images.githubusercontent.com/93148620/154567372-a73d3165-d221-4c1b-a5e5-7f3cbdb3c6ba.png)

При ДНК секвинировании можно заметить примерно одинаковое содержание нуклеотидов, что показано на графике:

![image](https://user-images.githubusercontent.com/93148620/154567126-e4fa66ac-7c41-4461-862e-3d6ac1f3cdbd.png)

Анализ QC прочтений проводился для образца 8 Cell (SRR5836473), тк для него уровень метилирования небольшой.Таким образом, учитывая небольшой уровень метилирования образца, часть метилированных (неизменных) цитозинов будет маленькой.

Анализ Per sequence GC content показывает, что содержание GC уменьшается. Так как часть цитозинов (неметилированные цитозины) в ходе бисульфидного секвинирования заменяется на урацилы, содержание цитозинов снижается:
